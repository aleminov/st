# Квиз по стяжке пола в Москве (Vue 3 + Vite)

Простой одностраничный сайт с квизом на 5 шагов по услуге стяжки пола в Москве.  
В конце квиза собираются контакты клиента (имя, телефон, комментарий, согласие на обработку данных).

## Запуск проекта локально

1. Установите зависимости:

```bash
npm install
```

2. Запустите dev-сервер:

```bash
npm run dev
```

3. Откройте в браузере адрес, который покажет Vite (обычно `http://localhost:5173`).

## Продакшн-сборка

```bash
npm run build
```

Готовые файлы появятся в папке `dist/` — её можно деплоить на любой статический хостинг (GitHub Pages, Vercel, Netlify и т.д.).

## Структура

- `src/App.vue` — общая рамка сайта, шапка/мобильное меню, модальное окно, `<RouterView>`.
- `src/components/FloorScreedQuiz.vue` — компонент квиза c 5 шагами и валидацией.
- `src/pages/*` — контент отдельных страниц (главная с квизом, услуги, объекты, цены, калькулятор, контакты).
- `src/router/index.js` — маршруты с ЧПУ (`/kalkulyator-styazhki`, `/nashi-uslugi` и т.д.).
- `src/main.js` — точка входа и подключение Vue Router.

## SEO

Сайт оптимизирован для поисковых систем:
- Уникальные title и description для каждой страницы
- Open Graph теги для социальных сетей
- Автоматическое обновление метатегов при навигации
- Семантическая разметка и ЧПУ URLs

При необходимости квиз можно доработать: добавить отправку данных на ваш backend в методе `submitForm` компонента `FloorScreedQuiz.vue`.
